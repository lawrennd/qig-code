Quantum Inaccessible Game (qig)
================================

**qig** is a Python package for studying quantum information geometry and the inaccessible game framework for constrained quantum dynamics.

The package provides tools for:

* **Quantum exponential families** on multi-qudit systems
* **Fisher information metrics** (BKM metric) for quantum states
* **Constrained dynamics** with information-geometric structure (GENERIC)
* **Entanglement analysis** using pair-operator bases
* **High-precision quantum derivatives** via Duhamel integration
